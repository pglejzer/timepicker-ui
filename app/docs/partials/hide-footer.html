Compact Wheel — Hide Footer

Footer completely removed from DOM — no OK/Cancel buttons rendered. Combine with commitOnScroll for a minimal UI.

HTML

<input id="nofooter-basic" value="02:30 PM" />
<input id="nofooter-crane" value="02:30 PM" />
<input id="nofooter-crane-straight" value="02:30 PM" />
<input id="nofooter-m3-green" value="02:30 PM" />
<input id="nofooter-dark" value="02:30 PM" />
<input id="nofooter-m2" value="02:30 PM" />
<input id="nofooter-glassmorphic" value="02:30 PM" />
<input id="nofooter-pastel" value="02:30 PM" />
<input id="nofooter-ai" value="02:30 PM" />
<input id="nofooter-cyberpunk" value="02:30 PM" />

JavaScript

const themes = [
  'basic', 'crane', 'crane-straight',
  'm3-green', 'dark', 'm2',
  'glassmorphic', 'pastel', 'ai', 'cyberpunk'
];

themes.forEach((theme) => {
  new TimepickerUI(`#nofooter-${theme}`, {
    ui: { mode: 'compact-wheel', theme },
    wheel: { hideFooter: true, commitOnScroll: true },
  }).create();
});

How it works

  • wheel.hideFooter: true — footer is not rendered in the DOM (not display:none — completely absent for accessibility)
  • Only works with compact-wheel mode
  • Combine with wheel.commitOnScroll: true so the selected time auto-commits without buttons
  • Click outside or press Escape to close the picker
