## Reporting a Vulnerability

If you discover a security vulnerability in **timepicker-ui**, please report it responsibly.

### How to report

- **Do not open a public GitHub issue**
- Send a report via:
  - GitHub **Security Advisories**, or
  - Email: **glejzerpiotr@gmail.com** *(recommended)*

### What to include

Please provide:
- A clear description of the issue
- Steps to reproduce (if possible)
- Affected version(s)
- Any relevant proof-of-concept or screenshots

### Response timeline

- Initial response: **within 72 hours**
- Status update: **within 7 days**

If the vulnerability is accepted:
- A fix will be released in a patched version
- A security advisory may be published

If the report is declined:
- You will receive a clear explanation
